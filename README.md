🧑‍💻 User Management Interface
This project demonstrates a simple User Management System built with a web interface.
It allows users to be added with their name and email address, and displays them in a dynamic list.

Features
✨ Add new users via a form with input fields for name and email.

📋 Display a list of all added users in real time.

🔄 Shows how form submissions update the user list.

⚠️ Currently, duplicate entries are not restricted (as seen in the screenshot), making it a good starting point for implementing validation logic.
